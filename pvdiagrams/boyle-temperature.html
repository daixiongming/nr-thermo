If we define y = pv and use the equation of the real gases, we obtain:

y3-bp+RTy2+apy-abp2=0

which will allow us to obtain the pv-p curves for a given temperature. The equation:

by2-ay+2ap=0

corresponds to the curve that connects the minimum values of the pv-p curve.

Library functions (Numerical Recipes):

zrhqr():
obtains the roots of a polynomial with real coefficients using the method of the eigenvalues. Its arguments are the vector a[] (which contains the coefficients of the polynomial), the complex vector rt[] (which contains the real and imaginary solutions).

The files for this calculation are in folder pvdiagrams/3_boyle/.
